# Анализ и предобработка датасета сетевого трафика

Добро пожаловать на сайт проектной работы.

Проект посвящен исследованию набора данных сетевого трафика, анализу его структуры, очистке и подготовке для последующего применения методов машинного обучения.

## Цель проекта

Изучение и обработка датасета сетевого трафика для обнаружения сетевых атак.

## Основные этапы:

- изучение структуры данных;
- поиск проблем;
- обработка NaN;
- удаление ошибок;
- анализ признаков;
- визуализация;
- подготовка данных.
