# Проект: Анализ и предобработка датасета сетевого трафика для обнаружения кибератак

## Описание проекта

В рамках проектной деятельности была выполнена работа по исследованию и анализу набора данных сетевого трафика с целью подготовки данных для дальнейшего обнаружения и классификации сетевых атак.

Современные информационные системы ежедневно подвергаются большому количеству угроз: DDoS-атакам, сканированию портов, ботнет-активности, попыткам несанкционированного доступа и другим видам вредоносной активности. Для своевременного обнаружения подобных событий применяются системы анализа сетевого трафика и методы машинного обучения.

Основной задачей проекта являлось исследование структуры датасета, анализ его параметров, очистка и предобработка данных, а также подготовка набора для последующего использования в системах обнаружения вторжений.

---

## Цель проекта

Провести исследование датасета сетевого трафика, выполнить предобработку данных, устранить ошибки и пропуски, проанализировать структуру признаков и подготовить данные для дальнейшего анализа и построения моделей обнаружения сетевых атак.

---

## Задачи проекта

Для достижения цели были поставлены следующие задачи:

- изучение структуры датасета;
- анализ признаков сетевого трафика;
- изучение типов сетевых событий;
- исследование наличия пропущенных значений;
- поиск аномалий и ошибочных данных;
- обработка значений NaN и бесконечностей;
- очистка датасета;
- анализ статистических характеристик;
- визуализация данных;
- подготовка информации для публикации на сайте проекта.

# Исследование структуры датасета

Для анализа использовался набор данных сетевого трафика, содержащий информацию о различных сетевых соединениях и их характеристиках.

Датасет включал более 80 признаков, описывающих:

- параметры пакетов;
- длительность соединений;
- количество переданных данных;
- временные характеристики;
- статистику сетевого потока;
- флаги TCP;
- признаки активности соединений.

Примеры признаков:

- Dst Port
- Protocol
- Flow Duration
- Flow Bytes/s
- Packet Length
- Flow Packets/s
- Active Mean
- Idle Mean
- Label

Отдельный столбец Label содержал информацию о типе трафика.
