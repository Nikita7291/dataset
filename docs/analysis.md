- Total Backward Packets
- Flow Bytes/s
- Flow Packets/s
- Packet Length Mean
- Active Mean
- Idle Mean

# Обнаруженные проблемы

В ходе исследования были обнаружены критические проблемы, способные повлиять на качество дальнейшего анализа.

## Проблема 1. Пропущенные значения NaN

Во многих столбцах присутствовали пропущенные значения.

Причины возникновения:

- ошибки формирования сетевых потоков;
- отсутствие некоторых параметров;
- некорректная обработка данных;
- ошибки при объединении файлов.

Наличие NaN приводит к следующим последствиям:

- невозможность обучения моделей;
- ошибки вычислений;
- снижение точности анализа;
- искажение статистики.

Для поиска пропусков использовалась команда:

```python
df.isnull().sum()
```

После проверки было установлено наличие большого количества пустых значений.

---

## Проблема 2. Бесконечные значения

Дополнительно были найдены значения:

```python
inf
-inf
```

Они возникали вследствие деления на ноль при вычислении характеристик потока:

Flow Bytes/s

Flow Packets/s

Подобные значения приводят к ошибкам:

- переполнению;
- сбоям обучения;
- неверным графикам;
- невозможности вычисления статистики.

---

## Проблема 3. Ошибочные типы данных

Некоторые столбцы содержали значения, не соответствующие ожидаемому типу.

Для решения применялось:

```python
pd.to_numeric(errors='coerce')
```

Некорректные значения автоматически заменялись на NaN.

---

## Процесс очистки данных

Очистка проводилась поэтапно.

### Шаг 1

Замена бесконечностей:

```python
import numpy as np

df.replace([np.inf,-np.inf],np.nan,inplace=True)
```

### Шаг 2

Преобразование признаков:

```python
for col in df.columns:
    df[col]=pd.to_numeric(df[col],errors='coerce')
```

### Шаг 3

Удаление поврежденных записей:

```python
df=df.dropna()
```

После очистки был сформирован новый датасет.
